# System Requirements

Phoenix Engine targets modern mid-range PCs. These requirements describe hardware for a comfortable gameplay experience, not the absolute lowest hardware that might launch the executable.

Very old PCs, legacy integrated GPUs, unsupported Vulkan drivers, and outdated office/laptop hardware are not supported.

## Windows

### Minimum Comfortable

- OS: Windows 10 or Windows 11, 64-bit.
- CPU: modern 4-core x64 processor.
- RAM: 8 GB.
- GPU: dedicated Vulkan 1.2-capable GPU, or a modern integrated GPU with current Vulkan drivers.
- VRAM: 2 GB.
- Storage: SSD strongly recommended.
- Driver: current NVIDIA, AMD, or Intel graphics driver.

Example GPU class: GeForce GTX 960 2 GB, GeForce GTX 1050, Radeon RX 460/RX 560, Intel Iris Xe, or better.

### Recommended

- OS: Windows 10 or Windows 11, 64-bit.
- CPU: modern 6-core x64 processor.
- RAM: 16 GB.
- GPU: Vulkan 1.2-capable dedicated GPU.
- VRAM: 4 GB minimum, 6 GB or more preferred.
- Storage: SSD.
- Driver: latest stable graphics driver.

Example GPU class: GeForce GTX 1660, GeForce RTX 2060/3050, Radeon RX 580, Radeon RX 5500 XT, Radeon RX 6600, or better.

## Linux

### Minimum Comfortable

- OS: modern 64-bit Linux distribution.
- CPU: modern 4-core x64 processor.
- RAM: 8 GB.
- GPU: Vulkan 1.2-capable GPU with a working ICD loader.
- VRAM: 2 GB.
- Storage: SSD strongly recommended.
- Driver: current Mesa RADV/ANV driver or current proprietary NVIDIA driver.

Example GPU class: GeForce GTX 960 2 GB, GeForce GTX 1050, Radeon RX 460/RX 560, Intel Iris Xe, or better.

### Recommended

- OS: modern 64-bit Linux distribution.
- CPU: modern 6-core x64 processor.
- RAM: 16 GB.
- GPU: Vulkan 1.2-capable dedicated GPU.
- VRAM: 4 GB minimum, 6 GB or more preferred.
- Storage: SSD.
- Driver: current Mesa RADV/ANV driver or current proprietary NVIDIA driver.

Example GPU class: GeForce GTX 1660, GeForce RTX 2060/3050, Radeon RX 580, Radeon RX 5500 XT, Radeon RX 6600, or better.

## Unsupported Hardware

- GPUs without Vulkan support.
- Vulkan 1.0/1.1-only hardware or drivers.
- Very old Intel HD Graphics generations.
- Windows 7-era machines.
- 32-bit operating systems.
- Systems with outdated or vendor-abandoned graphics drivers.

Phoenix Engine may run on some hardware below these recommendations, but those configurations are outside the supported target.
